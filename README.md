# 🎮 Guess Number Game

A mini project Python: Game **Guess the number** with 2 versions:
- `guess_number.py` → run in terminal.
- `gui_guess.py` → run in GUI (Tkinter).

---

# 🚀 How to run program

# Clone repo
```bash
git clone https://github.com/<username>/guess_number.git
cd guess_number
```
# Run program
## Play in terminal
```bash
py guess_number.py
```
## Play with GUI Tkinter
```bash
py gui_guess.py
```
