# 🎮 Guess Number Game

Một mini project Python: trò chơi **đoán số** với 2 phiên bản:
- `guess_number.py` → chạy trên terminal.
- `gui_guess.py` → giao diện GUI (Tkinter).

---

## 🚀 Cách chạy dự án

### 1. Clone repo
```bash
git clone https://github.com/<username>/guess_number.git
cd guess_number
```
## Run program
# Play in terminal
```bash
py guess_number.py
```
# Play with UI Tkinter
```bash
py gui_guess.py
```
